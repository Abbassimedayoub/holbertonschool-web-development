# Web Development — Landing Page Integration

## Overview

Integration of a complete landing page from a designer mockup, built section by section with **semantic HTML and CSS only**. The page includes a header, banner, quotes, video list, membership, FAQ and footer sections.

## Structure

| Directory | Content |
|---|---|
| `css_advanced/` | The full landing page: HTML structure and advanced CSS styling |
| `html_advanced/` | Semantic HTML groundwork for the page |

## Technologies

HTML5, CSS3 — no JavaScript, no framework.

## Skills Demonstrated

Translating a visual design into faithful, semantic markup; structuring CSS for a multi-section page; iterative, section-by-section delivery.
